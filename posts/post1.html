Post #1: Introducción a las Tablas Hash

¿Qué es una Tabla Hash?

Una tabla hash es una estructura que almacena pares clave–valor y usa una función hash para mapear claves a índices en un arreglo, permitiendo accesos muy rápidos.

Conceptos clave

  • Clave (key): Identificador único usado para buscar el valor.
  • Valor (value): Datos asociados a la clave.
  • Función hash: Convierte la clave en un número (índice).
  • Índice: Posición en el arreglo donde se almacena el valor.
  • Colisiones: Cuando dos claves producen el mismo índice.

¿Por qué son tan eficientes?

Operaciones promedio: O(1) para insertar, buscar y eliminar, porque la función hash calcula directamente la posición. En el peor caso (muchas colisiones) la complejidad puede degradar.
