Post #2: Manejo de Colisiones en Tablas Hash

Dos estrategias comunes

1) Encadenamiento (Chaining)

Cada posición del arreglo contiene una lista (o estructura enlazada) de pares clave–valor. Es fácil de implementar y crece dinámicamente.

2) Direccionamiento Abierto (Open Addressing)

Cuando ocurre una colisión, se busca otra celda libre en el arreglo usando una estrategia: Linear probing, Quadratic probing, Double hashing.

Métodos:

  • Linear probing: buscar la siguiente celda libre secuencialmente.
  • Quadratic probing: usar una función cuadrática para saltos (reduce clustering primario).
  • Double hashing: usar una segunda función hash para calcular el salto.
